Machine Learning Based Phishing Website Detection

Project Implementation

How our machine learning model detects phishing websites

Dataset

The dataset used contains more than 5 lakh URLs labelled as phishing or legitimate. The model learns patterns from these URLs and predicts if a new URL is malicious.

Feature Extraction

We extract 50 URL features such as:
• URL Length
• Number of dots
• Digit ratio
• Suspicious keywords
• HTTPS presence
• Subdomain count

Algorithms

The system was trained using advanced machine learning models:

• Random Forest
• XGBoost

Random Forest achieved the best accuracy and is used for prediction.

Model Evaluation

Performance analysis of the trained machine learning model

Confusion Matrix

Shows how many URLs were correctly and incorrectly classified by the model.

ROC Curve

The ROC curve measures how well the model distinguishes between phishing and legitimate URLs.

Performance Metrics

Accuracy, Precision, Recall and F1 Score are used to evaluate the machine learning model performance.
